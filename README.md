# Market Metrics

A clean, light-weight quantitative financial analytics tool written in Python. This repository serves as a public portfolio for financial technology development, focused on automating market data analysis and calculating core financial indicators.

## Features
*   **Simple Moving Average (SMA):** Calculates the lagging trend indicator for asset price series over defined windows.
*   **Volatility Analysis:** Computes the standard deviation of daily returns to quantify asset risk.
*   **Execution Ready:** Includes sample market data for immediate demonstration.

## Tech Stack
*   **Language:** Python 3.10+
*   **Key Libraries:** `numpy`, `pandas`
